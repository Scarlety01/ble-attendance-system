# BLE Smart Attendance System

BLE технологид суурилсан ухаалаг ирц бүртгэлийн систем.

## Төслийн тухай

Энэхүү систем нь BLE beacon, RSSI distance estimation, mobile application, backend API болон PostgreSQL өгөгдлийн сан ашиглан ирц бүртгэлийг автоматжуулах зорилготой.

## Үндсэн боломжууд

- BLE beacon илрүүлэх
- RSSI утгаар зай тооцоолох
- Auto check-in / check-out
- Оюутан, багш, админ хэрэглэгчийн эрхийн зохицуулалт
- FastAPI backend API
- PostgreSQL өгөгдлийн сан
- Redis offline queue / sync логик
- JWT authentication
- JMeter API load testing

## Ашигласан технологи

- Flutter / Dart
- FastAPI / Python
- PostgreSQL
- Redis
- Docker
- REST API
- JWT
- JMeter

## Төслийн зорилго

Мобайл апп болон BLE технологи ашиглан уламжлалт ирц бүртгэлийн гар ажиллагааг багасгаж, ирцийн мэдээллийг илүү найдвартай, хурдан, автомат байдлаар бүртгэх системийн prototype боловсруулах.
